Tipos de Almacenamiento de Datos

  • Almacenamiento de datos en la nube
  • Almacenamiento de datos en red
  • Almacenamiento de datos de objeto
  • Almacenamiento de datos de archivo
  • Almacenamiento de datos por bloque
  • Almacenamiento de datos definido por software

Nombre del Almacenamiento Caracterìsticas Principales Ejemplos Imagen
Datos en la Nube Consiste en resguardar datos en los servidores de un proveedor de servicios, gracias a la transmisión de datos por internet
  • Google Drive
  • Dropbox
  • ICloud
  • Soundcloud
Datos en Red Cuenta con una gran velocidad de transmisión de datos en una red
  • Storage Area Network (SAN)
  • Network Attached Storage (NAS)
  • Direct Attached Storage (DAS)
Datos de Objeto Resguarda datos sin ningún tipo de jerarquía o clasificación por carpetas. Se encarga de procesar los datos como unidades diferenciadas u objetos en un solo nivel de agrupación
  • Anàlisis de macrodatos
  • Internet de las cosas
  • Copia de seguridad
Datos de Archivo Se basa en una jerarquía de árbol, que dentro de una primera carpeta (el identificador único) incluye más carpetas. Funciona como el explorador de archivos de una computadora
  • Dropbox
  • Claro Drive
  • Google Drive
  • Amazon Drive
Datos por bloque Consiste en dividir tus datos en grupos y cada grupo contará con un identificador único. Se usa para organizar los datos, apartados unos de otros; pero al ser solicitados por el usuario se ensamblan de nuevo sin ningún problema
  • Contenedores
  • Cargas de trabajo transaccionales
  • Màquinas virtuales
Datos por Software Se encarga de desvincular el software de almacenamiento de los dispositivos (SDS), lo que a su vez permite una mayor flexibilidad y escalabilidad. Así es posible acceder a los datos con un único centro para administrarlos y automatizarlos
  • Google Drive
  • Dropbox
  • WeTransfer
